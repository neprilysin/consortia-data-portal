# Logo files

This demo currently uses styled placeholder logo cards.

To use real logos, place files here, for example:

- edinburgh.png
- strathclyde.png
- crh-cruk.png

Then replace the placeholder logo-card blocks in `app/templates/index.html` with image tags such as:

```html
<img src="/static/logos/edinburgh.png" alt="The University of Edinburgh">
<img src="/static/logos/strathclyde.png" alt="University of Strathclyde">
<img src="/static/logos/crh-cruk.png" alt="CRH-CRUK">
```
